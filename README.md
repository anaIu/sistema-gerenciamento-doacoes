# Sistema Interno de Gerenciamento de Doações

## Descrição

Sistema web desenvolvido para auxiliar instituições no gerenciamento interno de doações, permitindo o controle de doadores, doações, usuários e movimentações de estoque.

## Tecnologias

- Node.js
- Express
- PostgreSQL
- HTML
- CSS
- JavaScript

## Estrutura do Projeto

- public
- routes
- controllers
- models
- database

## Funcionalidades

- Login de usuários
- Gerenciamento de usuários
- Gerenciamento de doadores
- Registro de doações
- Controle de estoque
- Histórico de movimentações
- Dashboard
